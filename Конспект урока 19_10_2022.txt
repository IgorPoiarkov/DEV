MySQL база данных использует язык SQL - один из языков запросов.
В базе даннных у каждой записи есть id(Обычно проставляется системой автоматически). Остальны еполя вводятся по жеданию, на усмотрение создателя.
Select испльзуется при обращение скрипта к базе данных: Select *(name, surname) from users where user_id = 1.
